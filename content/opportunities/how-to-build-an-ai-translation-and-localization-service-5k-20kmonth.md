## The Workflow: Step-by-Step With Every Shortcut

### Step 1: Intake & Asset Extraction (10‑15 min)

Create a Notion page for every new job. Drop a simple form with name, deadline, source file link, target language, tone flag. Zapier hooks the form to a Google Drive folder; every file lands in the right place instantly.  

Pull the file into a Replit script that scrapes the raw text. Use `python-docx` for Word, `PyPDF2` for PDFs. Export plain‑text chunks no longer than 500 words. This keeps the OpenAI context window happy and cuts the time you’d spend manually copying content. Save the chunks back into the same Drive folder with a `source.txt` naming convention.  

> ⚡️ **HACK:** Store a small “translation memory” in a Google Sheet. Every time a chunk gets translated, paste the source–target pair. Later runs can call the sheet via Google Sheets API and skip duplicate phrases, saving you $0.02 per 1k tokens on GPT‑4.

### Step 2: AI Translation & QA (30‑45 min)

Run the Replit script again, this time calling ChatGPT’s GPT‑4 (around $20/month). Prompt:  
```
You are a professional translator. Translate the following text into Spanish, preserving the informal tone and idiomatic expressions.  
Text: [insert chunk]  
Output: Spanish text only.
```
Batch the calls via Make.com (5 $ plan) so every chunk spawns a new OpenAI request automatically. After the batch completes, pipe the output through Grammarly’s API (free tier limits 20 k words/month) for a final grammar jam.  

If the piece contains voice or audio, feed the Spanish text to ElevenLabs (20 $ plan) to synthesize a native‑sounding narrator. Save the MP3 alongside the text and update the Google Sheet with a “QA flag” if the grammar score dips below 95 %.  

> ⚡️ **HACK:** Add a “confidence filter” to your Make.com workflow. If GPT‑4’s token usage exceeds 2 k tokens for a 500‑word chunk, automatically re‑prompt with a stricter “avoid filler” instruction. That keeps the output tight and cuts token waste.

### Step 3: Localization & Cultural Adaptation (45‑60 min)

Pull the approved Spanish text into Canva for image overlay. Replace English captions with localized ones, using Canva’s brand kit to keep colors consistent. For brand‑new visuals, call Midjourney (subscription $10/month)
