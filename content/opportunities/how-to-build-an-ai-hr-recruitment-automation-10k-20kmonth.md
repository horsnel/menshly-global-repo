## The Workflow: Step-by-Step With Every Shortcut

### Step 1: Build the Resume Intake Pipeline (3 hrs)

First thing, you need a place to catch every resume that lands in your inbox, on LinkedIn, or in an ATS. Set up a Google Sheet as the hub—$0. This sheet will be the source of truth for all downstream tasks.

1. **Make.com** (free tier) – Create a scenario that watches a Gmail label “HR‑Inbox” for any attachment that ends in .pdf or .doc. When it finds one, it triggers a “Create Row” action in your Google Sheet.  
2. **Replit** – Spin up a tiny Python script on the free tier that pulls the raw PDF, uses *pdfplumber* to extract text, and uploads that text to a new column in the Sheet. You can drop the script into a Replit repo and schedule it with Replit’s “cron” feature (free tier).  

Now every candidate is logged in the Sheet with a unique ID, name, and resume text. This is the backbone of your automation; treat it like a database.

> **HACK:** If you’re on a tight budget, replace Make.com with **Zapier**’s free tier; the “Email by Zapier” action will do the same job. Just swap the trigger, keep the rest intact.

### Step 2: AI Screening Engine (4 hrs)

You’ve got the data, now let the AI decide who’s worth a call. Push the resume text to **ChatGPT 4** via its API. If you use the free tier of OpenAI’s API, you’re paying $0.0025 per 1k tokens. A typical résumé is ~1,200 tokens, so that’s only $0.003 per screening.

**Prompt Template:**  
```
You’re a hiring specialist for a SaaS startup. Rate this resume on a scale of 1–10 for fit, and list the top 3 red flags. If the score is 7+ and no red flags, mark “Go to Next Step.” Resume: [paste resume text]
```
Set the temperature to 0.2 to keep the output deterministic, and max tokens to 300 to avoid extraneous text.

After you hit the API, parse the JSON response and flag:
- **Score ≥ 8** → “Proceed” in a new column.
- **Score < 8** → “Hold” and note the reason.

Now you’re filtering candidates in under a second.

> **HACK:** For a no‑code version, plug the same prompt into **Make.com**’s “OpenAI” module; it automatically handles the API key and token limits. Save yourself the coding time.

### Step 3: Candidate Outreach & Scheduling (2 hrs)

You’ve got a shortlist. Time to contact and book interviews. Use **ActiveCampaign** (starts at $20/month) for email sequences and **Calendly** (free tier) for scheduling. Connect them via **Zapier** (free tier) or **Make.com**.
