### Step 2: Data Cleaning & Enrichment (1 h)  
Put the raw data into **Replit** to script a quick clean.  
Create a new Replit environment with Python 3.9, add `pandas` and `openpyxl`.  
Load the CSV, drop rows where `quantity == 0`, convert `order_date` to datetime, and fill missing SKUs with “Unknown SKU”.  
Here’s the snippet:

```python
import pandas as pd
df = pd.read_csv('orders.csv')
df = df[df['quantity'] > 0]
df['order_date'] = pd.to_datetime(df['order_date'])
df['sku'] = df['sku'].fillna('Unknown SKU')
df.to_csv('orders_clean.csv', index=False)
```

Save the file and run.
